# Contributing

Thanks for taking an interest in this repository.

## Scope

This repository is primarily a personal OpenClaw workspace, but contributions, ideas, and cleanup suggestions are still welcome.

## Guidelines

- Keep changes focused and easy to review
- Prefer small commits with clear messages
- Avoid committing noisy runtime artifacts unless explicitly needed
- Update documentation when behavior or structure changes
- Preserve workspace-specific conventions in `AGENTS.md`, `SOUL.md`, and skill folders

## Suggested workflow

1. Fork the repository
2. Create a feature branch
3. Make your changes
4. Run your local checks
5. Open a pull request with a clear summary

## Good contribution examples

- README/documentation improvements
- skill cleanup and refactoring
- dashboard UX improvements
- bug fixes in scripts
- safer defaults for generated/runtime outputs
